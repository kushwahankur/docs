>>> Why?
- NamedFormula is used to return different data types of values by using object attributes, and we can use named formula to fetch data from the list.
Here below a few  attributes which we can use in the named formulas:
* Logical operator(Mathematical, Logical, and Special Case).
* Functions (Date duration, Breakdown, String Treatment, Numerical function etc.)
* Constant value like string and numbers.

>>> When?
- Namedformula is using to retrieve data from the list or equivalent to inserting a formula in a table or creating user attributes with computed value.
here below a few conditions where we can use the named formulas:
* To sort string.
* To find data in the list of data.
* To modify exiting values.
* To find string length and lower and upper case conversion, substiute.
* To find absolute values, MAX and MIN values, and random numbers.
